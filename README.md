# Chromalchemy

Play here: https://isaacfhu.github.io/chromalchemy/

**Chromalchemy** is an elemental alchemy-style puzzle game, but with colors!
Starting with the 3 primary colors (Red, Green, Blue), drag and combine hues on the workspace to combine them and unlock hundreds of real named colors.

## Features

- **1566 Items to discover**
- **Drag and Drop interface**
- **Sound Effects**
- **Automatic Saving**
- **Color naming engine**
- **Custom theme (+Dark mode)**

## Tech Stack

- **HTML5:** Semantic layout with native `<dialog>` modals and HTML5 Drag & Drop API.

- **CSS / Tailwind CSS v4 / DaisyUI 5:** Styled with modern CSS and responsive layouts.

- **Vanilla JavaScript:** ES6+ logic handling HSL/RGB math, state management, audio, and local storage.

- **External Library:**
  [`ntc.js`](https://chir.ag/projects/ntc/) (Name That Color) for color name matching.

## How to Play

1. Drag Red, Green, or Blue from the right sidebar onto workspace.

2. Drag on color block on top of another to merge them.

3. If the mixture produces a new shade, it will be added to your color inventory!

4. Try to discover all named colors to fill your progress counter at the bottom left.
